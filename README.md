## Compilation du projet

```sh
cd poly4

make
```

Et c'est tout !

## Comment jouer

> 1. Configurer **l'IP:Port** dans le fichier `src/client.c` & compiler le projet.
> 2. Lancer le serveur avec `bin/server`.
> 3. Pour chaque joueur: lancer le client via `bin/client`
